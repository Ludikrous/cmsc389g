Introduction to Go

Course Description

Ever wish C was a little bit more like Python? So did Google!

Go is a rapidly growing, compiled language developed by Google, and is used often in the industry. This course will cover the basics of Go, its eccentricities, shortcomings, and its applications. We will also briefly cover microservices, and technologies related to it.

By the end of this course, you'll be able to write a backend in Go and comfortably add proficiency in Go to your resume. The final project should also add to your portfolio!
